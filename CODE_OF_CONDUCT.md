# Code of Conduct

This project aims to maintain a professional,
respectful and collaborative environment.

## Our Standards

- Respect contributors.
- Be constructive.
- Be professional.
- Focus on improving the project.

## Unacceptable Behavior

- Personal attacks
- Harassment
- Offensive language
- Spam

## Enforcement

Project maintainers reserve the right to remove
contributions that violate this Code of Conduct.
